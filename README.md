  <span style="color:#44AAFF;">🏁 1. 什么是 F1？</span>

  F1 (Formula 1)，全称一级方程式锦标赛，是国际汽联（FIA）举办的最高等级的年度系列场地赛车比赛。
  它不仅是速度的极限挑战，更是人类尖端科技、团队策略与顶尖车手勇气的完美结合。

  <span style="color:#FFD700;">"方程式"的含义：</span>并不是指数学公式，而是指所有参赛车辆都必须严格遵守统一的规则（Formula）来设计和制造。

  📅 赛季与大奖赛 (Grand Prix)
  * 全球巡演：每年约有 20~24 站比赛，横跨五大洲。
  * 比赛周末：通常为期三天（周五练习赛 -> 周六排位赛 -> 周日正赛）。

  🎯 比赛日核心机制
  * 正赛：约 305 公里或 2 小时的极限冲刺。
  * 积分系统（前10名得分）：
      🥇 第1名：25分  |  🥈 第2名：18分  |  🥉 第3名：15分
      第4名：12分  | 第5名：10分  | 第6名：8分
      第7名：6分   | 第8名：4分   | 第9名：2分  | 第10名：1分
  * 加分项：跑出最快单圈且排名前十，额外获得 +1分！

  🏅 双重冠军争夺
  1. 车队总冠军 (Constructors')：看哪家车队整体实力最强。
  2. 车手总冠军 (Drivers')：看哪位车手是最强王者。
